# Common Patterns

## Qualification pattern

Confirm intent, gather minimum required context, and route toward the right next owner or system.

## Intake pattern

Collect structured details early, validate what matters, and reduce repetition before handoff.

## Follow-up pattern

Trigger outbound contact from a known event, capture the outcome, and create a next step when
automation does not resolve it.

## Callback pattern

When the workflow cannot complete in-call, create a deterministic return path rather than leaving
the interaction unresolved.
