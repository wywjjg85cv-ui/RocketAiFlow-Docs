# Container Monitoring

Where RocketAiFlow runs in containers, container monitoring is part of the core operational view.

RocketAiFlow can include the `cAdvisor` Grafana dashboard as part of its pre-provisioned dashboard
foundation, so teams have immediate runtime visibility.

## What cAdvisor helps you see

`cAdvisor` is used for container-level runtime and resource visibility.

It helps teams review:

- container runtime state
- resource usage pressure
- whether runtime conditions line up with platform issues

## When to use it

Use container monitoring when:

- the platform is deployed in containers
- runtime pressure may be affecting workflows
- performance changes do not look purely telephony-related
- you need to separate application logic issues from runtime capacity issues

## What it helps investigate

- container instability
- resource contention
- whether runtime health changed at the same time as live workflow degradation
